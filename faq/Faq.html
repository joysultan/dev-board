Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer-2: Difference between innerHTML, innerText, and textContent is innerHTML use when need to insert or retrieve HTML content. Use innerText when need only visible text (ignores hidden elements) and use textContent when need all text, including hidden text.

Question-3: What is event delegation in the DOM?

Answer-3: Event delegation is a pattern in JavaScript where a parent element handles events for its child elements instead of attaching event listeners to each child individually. This is possible because of event bubbling, where an event triggered on a child element propagates (bubbles) up to its ancestors.

Question-4: What is event bubbling in the DOM?

Answer-4: Event bubbling is a phase in the DOM event propagation where an event starts from the target element and moves upward to its parent elements until it reaches the document object.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer-5: We can create, add, and remove elements using JavaScript:

Create an Element:

let newDiv = document.createElement("div");
newDiv.textContent = "Hello, World!";

Add an Element:

document.body.appendChild(createElement);

Remove an Element:

let elementToRemove = document.getElementById("myElement");
elementToRemove.remove();
